User-space environments the fast way

Get started

PRoot wrapper

Dockie uses PRoot, a user-space implementation of "chroot", "mount --bind" and "binfmt_misc". This means that users don't need root privileges to use a directory as the new root filesystem or executing programs built for a different architecture.

Simple

Dockie provides an easy workflow to manager different filesystems, making it easier create, run and destroy different environments.

Image Hub

Dockie allows pulling images from the LXC image server and Docker Hub with more thousands of images available! A single command will get your preferred system up and running in a matter of seconds.
